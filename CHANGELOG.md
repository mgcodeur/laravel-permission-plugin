# Changelog

All notable changes to `laravel-permission-plugin` will be documented in this file.

## 1.1.0 - 2026-04-18

### What's Changed

#### Features

- Add Laravel 13 support (`illuminate/contracts: ^13.0`)
- Extend `orchestra/testbench` to `^11.0` for Laravel 13
- CI matrix now tests against Laravel 11, 12 and 13

#### Fixes

- Fix invalid `conflicts` key in `composer.json` (should be `conflict`) that prevented the package from being published

#### Compatibility

- PHP: 8.1 / 8.2 / 8.3 / 8.4
- Laravel: 11 / 12 / 13
- `spatie/laravel-permission`: ^6.21
